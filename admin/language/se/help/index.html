WebClient är utformad till att fungera precis som en windowsapplikation och stödjer

  • Högerklicka med musen för snabbmeny
  • Dra och släpp för att flytta, kopiera och skapa poster mellan mappar

  • Kortkommandon för snabba tillämpningar
  • Webbläsaraktiviteter(Uppdatera, Tillbaka, Framåt) loggar ut dig

E-posttips

  • För att skapa en mapp, välj en mapp, högerklicka - lägg till mapp.
  • För att ta bort flagga på ett meddelande, högerklicka & välj ta bort flagga eller tryck "-" på numpad.
  • För att skapa ett avtalat möte utan att lämna mappen, välj nytt - möte från huvudmenyn
  • För att sortera eller flagga meddelanden automatiskt, använd verktyg - regler.
  • För att skapa en PIM-post med e-postbilagor, dra & släpp e-mejl till en PIM-mapp
  • För att ladda upp en bilaga, dra den från skrivbordet till bilageavdelningen

Certifikat

  • Se till att ladda upp din privata nyckel (PKCS#12) i verktyg- alternativ - säkerhet.
  • För att signera ett meddelande väljer du knappen signera i det skapade meddelandet.
  • För att kryptera ett meddelande behöver du en publik certifikatsnyckel (PKCS#7) som mottagaren laddade upp i certifkatsfliken i kontakt.

Meddelanden dekrypteras med din privata nyckel (PKCS#12). Signaturvalidering görs via avsändarens publika certifikatsnyckel (PKCS#7) som nyligen laddats upp i kontakt.

Kalender

  • Klicka för att ange start tid/datum för ett nytt möte
  • Dra och håll för att ange sluttid/datum för mötet
  • Skriv en titel eller högerklicka - välj ny för att skapa möte
  • Högerklicka händelse för snabbmeny
  • Dubbelklicka händelse för att redigera eller välj och tryck F2
  • Dubbelklicka på dagfliken för att lägga till en heldagsaktivitet
  • Klicka och håll på händelsemarkören (markör visas) för att flytta aktiviteten
  • Klicka och håll händelsesidorna för att ändra start/slut tid/datum
  • För att gå framåt/bakåt till månad/vecka/dag, klicka på pilarna längst upp.
  • För att få meddelande om kommande händelser, ange påminnelse att visas som ett pop-up-fönster i dialogfönstret i WebClient, mobila enheter och andra applikationer.

Snabbmeddelanden

  • Klicka på statusikonen uppe i högra hörnet för att visa - Visa eller dölj IM
  • Ändra statusikonen uppe i vänstra hörnet av IM till online
  • Klicka på användarikonen för att dölja visning av endast användare online
  • Dra och släpp användare för att ändra gruppen
  • Använd kortikonen för att se användardetaljer och foto
  • Använd mina detaljer i verktygsmenyn för att redigera hur andra kan se dig

Delning

  • För att dela dina mappar, högerklicka på mappen och ange rättigheter och delning.
  • För att få tillgång till någons delade konto, högerklicka på huvudmappen och välj lägg till delat konto.

WebPhone

  • Kllicka på telefonikonen uppe i högra hörnet, välj alternativ för att konfiguera den
  • Använd telefonboksfilter "Visa SIP kontakter" för att lista endast internetnummer
  • Använd snabbmeddelandelistan och kontakter för att ringa direkt

Observera.
För att få WebPhone Java applet att fungera behöver du ha JRE (Java Runtime Environment) installerat.

SmartAttach

För att spara bandbredd och få kontroll över innehållet i skickade e-mejl kan du använda SmartAttach och ersätta e-postbilagor med hyperlänkar.
  • Skriv ett nytt e-mjel med bilagor och möjliggör alternativet SmartAttach i verktygsfältet
  • Bläddra igenom filmappar för att kunna se och hantera SmartAttached-filer som är lagrade i mappen
  • Många bilagor kan läggas till via Upload (diskfiler), poster (kontakter, anteckningar etc.) eller genom att dra & släpp

Textmeddelande (SMS)

  • Välj nytt SMS från verktygsmenyn (också under verktygsmenyn - alternativ - SMS när du skriver e-post)
  • Skriv ett telefonnummer eller klicka på bokikonen för att få fram mottagare med ett mobilnummer
  • Skriv ditt meddelande antingen i ämne eller huvudmeddelandet
  • Bekräfta meddelandet i förhandsgranskarutan längst ner

Återställningsposter

Detta fungerar på samma sätt som papperskorgen för e-postmeddelanden. Du kan se raderade kontakter, händelser, uppgifter, etc. Administratören har förinställt för hur många poster som kommer att hållas kvar efter radering.
  • Navigera till mappen för återställningsposter
  • Högerklicka på posten och du kan antingen återställa den till dess ursprungliga plats
  • eller du kan radera posten permanent
Tillbaka till toppen av sidan

Backup och återställning

Backup sker automatiskt av GroupWare-data på servern, men du kan skapa dina egna backuper som kan importeras till andra applikationer.
  • Verktyg - Export / Import har alternativ till backup och återställning av all GroupWare- data
  • eller du kan exportera hela kontaktmappen till olika format
  • I kontaktmappen, välj några poster, högerklicka och spara som vCard
  • I kalenderns listvy, välj några poster, högerklicka och spara som vCalendar
