Bien utiliser la partie Email

  • Pour créer un dossier, sélectionner le dossier parent, clic droit - Ajouter un dossiser.
  • Pour supprimer la coche d'un message, clic droit et Supprimer coche, utiliser la touche '-' du clavier numérique.
  • Pour créer un rendez-vous tout en restant dans le dossier mail, sélectionner Créer - Nouvel événement à partir du menu principal.
  • Pour classer ou pour marquer les messages automatiquement, utiliser Outils - Règles.
  • Pour créer un élément PIM avec le message en attachement, glisser le message dans un dossier de type PIM (ex. Calendrier, Tâches, Contacts).
  • Pour attacher une pièce jointe, glisser le depuis le bureau dans la barre de pièces jointes

Certificats

  • Assurez-vous d'avoir stocké votre clé privé (PKCS#12) dans Outils - Options - Certificats privés.
  • Pour signer un message, cocher "Signer" dans le ruban du menu "Options".
  • Pour crypter un message vous devez avoir stocké le certificat public (PKCS#7) du destinataire dans l'onglet Certificat du contact.

Les messages sont décryptés avec votre clé privé (PKCS#12). La signature d'un expéditeur est vérifiée en utilisant le certificat public (PKCS#7) de l'expéditeur stocké préalablement dans les propriétés du contact.

Contrôles calendrier

  • Cliquer pour fixer l'heure/la date de début d'un nouveau rendez-vous.
  • Garder appuyé et déplacer pour fixer l'heure/date de fin du rendez-vous.
  • Écrire le titre d'un rendez-vous ou clic droit sur une zone de sélection suivi de Nouveau pour créer un rendez-vous.
  • Clic droit sur le rendez-vous présente le menu contextuel.
  • Double cliquer sur le rendez-vous ou F2 pour l'éditer.
  • Double cliquer sur le titre d'une journée pour créer un nouveau rendez-vous pour la journée entière.
  • Dans la vue par mois, cliquer sur le rendez-vous et maintenir pour déplacer le rendez-vous.
  • Dans la vue par semaine/par jour, cliquer sur la bordure gauche du rendez-vous et maintenir pour déplacer le rendez-vous.
  • Cliquer sur les bordures haut, bas, droit et bas pour modifier l'heure de début/l'heure de fin/la date du rendez-vous.
  • Pour changer de mois/semaine/jour, utiliser les flèches ou les menus déroulant en haut du calendrier.
  • Pour recevoir des rappels par pop-up dans le Client Web, téléphones mobiles et d'autres applications, positionner l'option "Me rappeler" du rendez-vous.

Messagerie instantanée

  • Cliquer sur l'icône du statut IM en haut à droite pour afficher ou masquer la colonne IM.
  • Changer le statut IM en cliquant sur l'icône de gauche dans le titre de la colonne IM.
  • Cliquer sur Tous pour basculer entre "voir tous les contacts"/"ne voir que les contacts en-ligne".
  • Cliquer et déplacer pour changer le contact du groupe.
  • Utiliser l'icône flèche en haut et à droite pour déplier/replier le volet IM.
  • Utiliser Outils - Mes données pour éditer les informations vues par les autres.

Partage

  • Pour partager un dossier, clic droit sur le dossier et choisir Partage et Droits.
  • Pour visualiser des dossiers d'un autre utilisateur, clic droit sur le dossier racine et choisir Ajouter un compte partagé.
Haut

Téléphone web

  • Utiliser Options - Téléphone pour configurer le WebPhone
  • Utiliser clic droit sur un contact IM pour l'appeler
  • Utiliser le bouton "Appeler" pour des appels audio ou cliquer sur l'icône vidéo pour des appels vidéo

Note.
WebPhone n'est disponible qu'à partir d'un navigateur WebRTC (actuellement Chrome et Firefox). Un plugin est nécessaire sur Internet Explorer et Safari. En cas d'incompatibilité du navigateur, vous serez redirigé vers une page Web contenant des instructions de vérification et configurtaion.

SmartAttach

Pour éconmoiser la bande passante et pour mieux contrôler le contenu des mails sortants, utiliser la technologie SmartAttach pour remplacer les pièces jointes par des hyperliens.
  • Lors de la composition d'un nouveau message, ajouter les pièces jointes et ensuite cocher "Utiliser SmartAttach".
  • Consulter le dossier "Documents" pour gérer toutes les pièces jointes stockées dans ce dossier avec l'étiquette "SmartAttach".
  • Il est possible d'attacher plusieurs pièces jointes fichiers à partir d'explorateur; emails, contacts, notes... par Glisser - Déposer

Text Messaging (SMS)

  • Choisir Créer - Nouveau SMS à partir du menu principal.
  • Entrer un numéro de téléphone mobile or cliquer sur l'icône '+' pour choisir un contact parmi ceux avec un numéro de téléphone mobile.
  • Entrer le texte soit dans le champ Objet, soit dans le corps du message.
  • Le texte peut être vérifié dans la fenêtre de prévisualisation.

Restaurer les éléments supprimés

La corbeille Groupware contient les éléments supprimés des types: rendez-vous, contact, tâche... L'administrateur peut préciser le nombre jours durant lesquels les éléments seront conservés dans la corbeille.
  • Naviguer vers le dossier Corbeille groupware.
  • Clic droit sur l'élément de la corbeille, et vous pouvez soit restaurer l'élément dans son dossier d'origine,
  • ou le supprimer de manière permanente.
Haut

Sauvegarde et restauration

Même si les données Groupware peuvent être sauvegardées automatiquement sur le serveur, vous pouvez sauvegarder les données groupware pour les importer dans d'autres applications.
  • Pour sauvegarder/restaurer les données Groupware, utiliser Outils - Importer/Exporter,
  • ou vous pouvez exporter les dossiers contacts dans divers formats.
  • Dans le dosssier Contacts, slectionner les contacts, clic droit et Enr. format vCard.
  • Dans la vue liste d'un dossier Calendrier, sélectionner les éléments, clic droit et Enr. format vCalendar.
