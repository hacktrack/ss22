Sertifikaatit

  • Lähetä salainen avaimesi (private key) (PKCS#12) kohdassa Asetukset  - Private Certificates.
  • Allekirjoita viesti valitsemalla Allekirjoita uuden viestin luonnissa.
  • Salataksesi viestin tarvitset vastaanottajan julkisen avaimen (public key certificate) (PKCS#7). Lisää julkinen avain kontaktin tiedoissa välilehdellä Sertifikaatti.

Viestit puretaan käyttäen sinun salaista avainta (PKCS#12). Allekirjoitus varmistetaan käyttäen lähettäjän julkista avainta (public key certificate / PKCS#12), jonka olet lisännyt kontaktin tietoihin.

Pikaviestin

  • Click the status icon in upper right corner to Show or Hide IM

  • Change the users icon to Hide to show only online users

  • Use card icon to see user's details and photo
  • Use My Details in Tool menu to edit how others see you

WebPhone

  • Klikkaa puhelinkuvakket oikeassa yläkulmassa, valitse Asetukset ja konfiguroi se
  • Käytä puhelinluettelon suodatinta "Näytä SIP-kontaktit" listataksesi vain internetnumerot
  • Käytä pikaviestinrosteria ja kontakteja soittaaksesi suoraan

Huom:
Jotta WebPhone Java applet toimii, tulee JRE:n (Java Runtime Environmenti) olla asennettuna.

SmartAttach

Säästääksesi verkkoliikennettä ja kontrolloidaksesi paremmin lähetettyjen viestien sisältöä voit käyttää SmartAttachia korvaamaan liitetiedostot linkeillä.

  • Selaa Tiedostot-kansiota ja hallinnoi SmartAttachilla liitettyjä tiedostoja
  • Voit liittää useita tiedostoja käyttämällä raahaamista.

Palautettavat kohteet

Tämä toimii kuten sähköpostiviestien roskakori. Näet poistetut kontaktit, tapahtumat, tehtävät, yms. Ylläpitäjä on asettanut kuinka monen päivän päästä kansio tyhjennetään automaattisesti.
  • Siirry Palautettavat kohteet -kansioon
  • Oikea-klikkaa kohdetta ja voit joko palauttaa kohteen alkuperäiseen paikkaansa tai poistaa sen

Turvakopiot ja palauttaminen

GroupWaren tiedoista otetaan turvakopiot automaattisesti, mutta voit ottaa oman turvakopion ja vaikka viedä tiedot toiseen ohjelmaan.
  • Asetukset - Tuo/Vie sisältää valinnan ottaa turvakopion tai palauttaa tiedot
  • Voit viedä kaikki kontaktisi useissa eri muodoissa
  • Yhteystiedot kansiossa valitse kontakteja, oikea-klikkaa ja valitse Tallenna vCard-muodossa
  • Kalenterissa valitse tapahtumia, oikea-klikkaa ja valitse Tallenna vCalendar-muodossa
