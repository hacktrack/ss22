웹클라이언트는 완전한 기능을 갖춘 윈도우 기반 어플리케이션으로 설계되었습니다.

  • 컨텍스트 메뉴에 마우스 오른쪽 클릭
  • 드래그 & 드롭 메뉴로 폴더 간의 이동, 복사, 항목 생성
  • 데스크톱에서 드래그 & 드롭하여 파일 업로드
  • 바로 가기 키로 응용 프로그램 제어
  • 브라우저 제어 (리로드, 뒤로, 앞으로) 로그아웃

이메일 힌트

  • 폴더를 만들려면 폴더를 선택하고 마우스 오른쪽 단추로 - 폴더 추가를 누르십시오.
  • 메시지 플래그를 해제하려면 해당 메시지를 마우스 오른쪽 버튼으로 클릭하고 플래그 해제를 선택하거나 숫자판에 "-"를 누르십시오.
  • 메일 폴더를 남기지 않고 약속을 만들려면 메인 메뉴에서 [새로 만들기] - [예약]을 선택하십시오.
  • 메시지를 자동으로 정렬하거나 플래그를 지정하려면 [옵션] - [메일] - [규칙]을 사용하십시오.
  • 전자 메일 첨부 파일이 있는 PIM 항목을 만들려면 PIM 폴더로 전자 메일을 끌어다 놓으십시오.
  • 첨부를 업로드하려면 바탕 화면에서 끌어다 첨부 파일 표시줄에 놓으십시오.
맨 위로

인증서

  • 옵션 - 계정 - 개인 인증서에 개인 키(PKCS#12)를 업로드하십시오.
  • 메시지에 서명하려면 작성 창에서 서명 버튼을 선택하십시오.
  • 메시지를 암호화하려면 연락처의 인증서 탭에 업로드되거나 GAL(공용 연락처 폴더)에서 사용할 수 있는 수신인의 공용 키 인증서(PKCS#7)가 필요합니다.

메시지 암호는 개인키(PKCS#12)를 사용하여 해독됩니다. 서명 확인은 이전에 연락처에 업로드한 발신인의 공개 키 인증서(PKCS#7)를 통해 수행됩니다.

캘린더 컨트롤

  • 새 약속의 시작 시간/날짜를 설정하려면 클릭하십시오.
  • 보류 및 끌어서 약속의 종료 시간/날짜 설정
  • 약속 제목 쓰기 또는 오른쪽 클릭 선택 - 새로 만들기를 선택하여 약속 만들기
  • 콘텍스트 메뉴에 대한 마우스 오른쪽 버튼 이벤트
  • 이벤트를 더블클릭하여 편집하거나 선택하고 F2를 누릅니다.
  • 일자를 더블 클릭하여 새 종일 이벤트 추가
  • 이벤트바(핸들이 나타남)를 누르고 있으면 이벤트 이동
  • 시작/종료 시간/날짜를 변경하려면 이벤트 측면을 클릭하고 유지하십시오
  • 월/주/일별로 앞으로/뒤로 이동하려면 상단의 화살표를 클릭하십시오.
  • 향후 이벤트에 대한 알림을 받으려면 해당 이벤트에 대한 알림 확인란을 선택하여 웹클라이언트, 모바일 장치 및 기타 응용 프로그램에서 알림 팝업이 나타나도록 하십시오.

인스턴트 메시징

  • 오른쪽 위 모서리에 있는 둥근 아이콘을 클릭하여 IM 상태를 확인하거나 변경하십시오.
  • 오른쪽 위 모서리에 있는 둥근 아이콘을 클릭하여 IM과 함께 온라인 상태로 이동
  • 대화 목록의 맨 아래에 있는 사용자 아이콘을 클릭하여 온라인 상태인 사용자만 표시하십시오.
  • 사용자를 드래그 앤 드롭하여 그룹 변경
  • 맨 오른쪽 상단 모서리의 개인 이미지 위를 맴돌 때 열리는 통신 메뉴에서 내 세부 정보 사용

공유

  • 폴더 공유를 활성화하려면 해당 폴더를 마우스 오른쪽 단추로 클릭하고 공유 및 사용 권한을 클릭한 다음 원하는 사용자를 선택하십시오.
  • 다른 사용자의 공유 계정에 액세스하려면 루트 폴더를 마우스 오른쪽 버튼으로 누르고 공유 계정 추가를 선택하십시오.

웹폰

  • 옵션 선택 - 개인 이미지 위에 오른쪽 상단을 맴도는 후 WebPhone 설정
  • 오른쪽 클릭이나 연락처가 있는 인스턴트 메시징 레지스터를 사용하여 직접 통화하십시오.
  • 콜 버튼을 사용하여 오디오 전화를 걸거나 비디오 아이콘을 클릭하여 화상 통화를 하십시오.

메모
WebPhone에는 WebRTC 가능 브라우저(현재 크롬 및 파이어폭스) 또는 플러그인(Internet Explorer 및 Safari용)이 필요하다. 브라우저가 전화 통화를 처리할 수 없는 경우 도움말 페이지로 리디렉션됩니다.

대용량 첨부

대역폭을 절약하고 전송된 이메일의 내용을 제어하기 위해 대용량첨부를 사용하여 이메일 첨부 파일을 하이퍼링크로 바꿀 수 있습니다.
  • SmartAttach 버튼을 클릭하여 새 전자 메일을 구성하고 첨부 파일을 추가하십시오.
  • 파일 폴더를 탐색하여 폴더에 저장된 SmartAttached 파일을 보고 관리하십시오.
  • 여러 첨부 파일을 로컬 디스크에서 첨부하거나 WebClient(기존 연락처, 노트 등)에서 첨부하거나 끌어서 놓을 수 있습니다.

복구 항목

이것은 이메일 메시지에 대한 휴지통 폴더와 같은 기능을 합니다. 삭제된 연락처, 이벤트, 작업 등을 볼 수 있습니다. 관리자는 항목을 삭제한 후 보관할 일 수를 미리 설정할 수 있습니다.
  • 복구 항목 폴더로 이동
  • 항목을 마우스 오른쪽 단추로 클릭하면 원래 위치에 복원할 수 있습니다
  • 또는 영구적으로 항목을 삭제할 수 있습니다.
맨 위로

백업 및 복원

GroupWare 데이터는 서버에 자동으로 백업되지만, 다른 응용 프로그램에서도 가져올 수 있는 자신의 백업을 생성할 수 있습니다.
  • 옵션 - 내보내기/가져오기 옵션으로 모든 GroupWare 데이터 백업 및 복원 가능
  • 또는 폴더를 마우스 오른쪽 버튼으로 클릭하고 폴더 내보내기를 선택하여 전체 폴더를 내보낼 수 있습니다.
  • 연락처 폴더에서 일부 항목을 선택하고 마우스 오른쪽 버튼을 클릭한 후 vCard로 저장하십시오.
  • 일정관리 목록 보기에서 일부 항목을 선택하고 마우스 오른쪽 버튼을 클릭한 후 vCalendar로 저장하십시오.
