WebClient is designed as a full-featured window based application supporting

  • Right-click mouse for context menu
  • Drag & drop to move, copy and create items between folders
  • Drag files from desktop & drop to upload them
  • Keyboard shortcuts control the application
  • Browser controls (Reload, Back, Forward) log you out

Keyboard Controls

Up, Down Move in message list, scroll in message view
Enter Open highlighted message
Ctrl+Enter Open highlighted message in a new window
Shift+Up, Shift+Down Select series in list
Hold Shift+left click Select series in list
Hold Ctrl+left click (Un)select multiple messages
Ctrl+A Select all
End Jump to last item in list
Home Jump to first item in list
+ flag message ticked
- Unflag message
0..5 (keypad) Set color flag
r Reply
R Reply to all recipients
f Forward
F Forward as attachment
e mark Read
u mark Unread
c Copy to folder
m Move to folder
n New message compose window
Delete Delete
Esc Close window, No in Are you sure.. dialog

Note.
Keyboard shortcuts work only if message list is focused. Sometimes you need to click a message in the list first.

Email Hints

  • To create a folder, select a folder, right-click - Add Folder.
  • To unflag a msg, right-click it & select Unflag or press "-" on numpad.
  • To create an appointment without leaving mail folder, select New - Appointment from main menu.
  • To sort or flag messages automatically, use Options - Mail - Rules.
  • To create a PIM item with email attachments, drag & drop email to a PIM folder
  • To upload an attachment, drag it from desktop and drop it onto the attachments bar

Certificates

  • Make sure to upload your private key (PKCS#12) in Options - Accounts - Private Certificates.
  • To sign a message select the Sign button in the compose window.
  • To encrypt a message you need a public key certificate (PKCS#7) of the recipient uploaded in Certificate tab of the contact or available from a Public Contacts folders (GAL).

Messages are decrypted using your private key (PKCS#12). Signature validation is done via the sender's public key certificate (PKCS#7) previously uploaded in the contact.

Calendar Controls

  • Click to set start time/date of a new appointment
  • Hold and Drag to set end time/date of the appointment
  • Write an appointment title or Right click selection - select New to create appointment
  • Right-click event for context menu
  • Double-click event to edit or select and press F2
  • Double-click day label to add new all-day event
  • Click and hold event bar (handle appears) to move the event
  • Click and hold event sides to change start/end time/date
  • To go forward/backwards by month/week/day, click the arrows at the top.
  • To be notified of upcoming events, check the reminder checkbox for that event to have a reminder pop up in WebClient, mobile devices and other applications.

Instant Messaging

  • Click the round icon in upper right corner to see or change your IM status
  • Click the round icon in upper right corner to go Online with IM
  • Click the people icon down at the bottom of the chat list to show only users which are online
  • Drag and drop users to change the group
  • Use My Details in Communication menu, which opens when hovering over your personal image on the far upper right corner, to edit how others see you

Sharing

  • To enable sharing of your folders, right click the appropriate folder, click Sharing and Permissions and select the wished user(s).
  • To access somebody's shared account, right click the root folder and select Add Shared Account.

WebPhone

  • Choose Options - Phone after hovering by the top right over your personal image to configure WebPhone settings
  • Use the instant messaging roster with right clicking or contacts to make calls directly
  • Use the Call button to make audio calls or click the video icon to make video calls

Note.
The WebPhone requires a WebRTC-able browser (currently Chrome and Firefox) or plugin (for Internet Explorer and Safari). You will be redirected to a help page in case your browser can not handle phone calls.

SmartAttach

In order to save bandwidth and have control over the content of sent emails you can use SmartAttach to replace email attachments with hyperlinks.
  • Compose a new email and add attachments with SmartAttach by clicking the SmartAttach button after adding them
  • Browse through Files folder to see and manage SmartAttached files stored to the folder
  • Multiple attachments can be Attached from Local Disk, or Attached from WebClient (existing contacts, notes etc.) or by drag & drop

Text Messaging (SMS)

  • Select New - SMS from the menu bar
  • Type a phone number or click the Plus icon to select from recipients with a mobile number
  • Compose your message either to subject or body
  • Verify the message in Preview pane at the bottom

Recovery Items

This works the same as the Trash folder for email messages. You can see the deleted contacts, events, tasks, etc. The administrator has preset for how many days the items will be kept after deletion.
  • Navigate to Recovery Items folder
  • Right-click the item and you can either Restore them to their original place
  • Or you can Delete the item permanently
Back to Top

Backup and Restore

GroupWare data are backed up on the server automatically, but you can create your own backups that can be also imported in other applications.
  • Options - Export / Import has options to backup and restore all GroupWare data
  • Or you can export whole folders by right-clicking them and choosing Export Folder
  • In contacts folder, select some items, right-click and Save as vCard
  • In calendar List view, select some items, right-click and Save as vCalendar
