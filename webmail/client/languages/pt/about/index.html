O Cliente Web IceWarp é um conjunto completo de aplicativos web de produtividade, incluindo Email, Calendário, Documentos, Planilhas, Apresentações, TeamChat, bate-papo privado, chamadas de voz e vídeo e armazenamento seguro em storage.

Projetado para o século XXI. Desenvolvido usando as últimas tecnologias:
  • padrão HTML5 para experiência de desktop na web
  • WebRTC para áudio multiponto, streaming de vídeo e compartilhamento de tela
  • ONLYOFFICE um projeto desenvolvido pelo Ascensio System SIA
  • WeatherUnderground
  • Google Maps
  • WebAPI e TeamChatAPI para integrações personalizadas
