O Cliente Web foi concebido como um aplicativo Windows completo que suporta as seguintes capacidades:

  • Clique com o botão direito do mouse para visualizar o menu de contexto.
  • Clique, segure e arraste para mover, copiar e criar itens entre pastas.
  • Arraste arquivos da área de trabalho e solte-os para enviá-los.
  • Atalhos de teclado controlam o aplicativo (detalhes adiante).
  • Controles do navegador (Atualizar, Voltar, Avançar) fazem com que sua sessão seja finalizada, portanto, não os utilize.

Controles de Janela

  • Clique, segure e arraste as bordas das janelas para redimensioná-las.
  • Clique em [_] para minimizar a janela na área inferior de encaixe.
  • Clique em um item na área de encaixe para restaurá-lo.
  • Click no [x] para fechar a janela.
  • Clique duas vezes no canto superior direito para fechar uma janela.

Dicas de e-mail

  • Para criar uma pasta, selecione uma pasta, clique com o botão direito do mouse e escolha Adicionar pasta.
  • Para remover o sinalizador de uma mensagem, clique com botão direito, selecione Remover o sinalizador ou pressione "-" no teclado numérico.
  • Para criar um novo compromisso sem sair da pasta de e-mail, selecione Novo(a) - Compromisso no menu principal.
  • Para classificar ou sinalizar mensagens automaticamente, use Opções - Correio - Regras.
  • Para criar um item de PIM (item de GroupWare) com anexos de e-mail, arraste e solte o e-mail para uma pasta PIM.
  • Para enviar um anexo, arraste-o da área de trabalho e solte-o na barra inferior de anexos de um novo e-mail ou clique em Anexar do Disco Local.

Certificados

  • Certifique-se de fazer o upload da sua chave privativa (PKCS#12) em Opções - Contas - Certificados Privativos.
  • Para assinar uma mensagem, selecione o botão Assinar digitalmente a mensagem, na janela de composição de uma mensagem.
  • Para criptografar uma mensagem, você precisa de um certificado de chave pública (PKCS #7) do destinatário carregado na guia Certificados do contato.

Mensagens são descriptografadas usando sua chave privada (PKCS #12). A validação da assinatura é feita através certificado de chave pública do remetente (PKCS #7) previamente carregado no contato.

Controles do calendário

  • Clique para definir a data e hora de início de um novo compromisso.
  • Segure e arraste para definir a data e hora de término do compromisso.
  • Escreva um título de compromisso ou clique com o botão da direita numa área selecionada e escolha Novo(a), para criar um novo compromisso.
  • Clique com o botão direito do mouse para abrir o menu de contexto.
  • Dê um duplo clique no compromisso para editar ou pressione F2.
  • Dê um duplo clique no rótulo do dia para adicionar novo evento de dia inteiro.
  • Clique e segure a barra de um compromisso para movê-lo.
  • Clique no canto superior ou inferior direito da barra de um compromisso, segure e arraste para alterar a data/hora de início/término.
  • Para ir para frente/para trás por dia/semana/mês, clique nas setas na parte superior.
  • Para ser notificado sobre compromissos futuros, defina o lembrete para abrir uma janela de diálogo no Cliente Web, em dispositivos móveis e em outras aplicações.

Mensagens Instantâneas

  • Clique no ícone circular, na parte super direita, para alterar seu status.
  • Clique no ícone circular, na parte super direita, para ficar Online.
  • Clique no ícone de duas pessoas, na parte inferior da listagem do chat, para ocultar/exibir apenas usuários online
  • Arraste e solte usuários para alterar o grupo.
  • Use a opção Meus Detalhes, na parte superior direita, abaixo da foto do seu usuário, para editar como outros usuários visualizam suas informações, caso compartilhadas.

Compartilhamento

  • Para habilitar o compartilhamento de pastas, clique com botão direito em qualquer pasta, selecione Compartilhamento e Permissões e selecione o(s) usuário(s) desejado(s).
  • Para acessar a conta compartilhada de alguém, clique com o botão direito na pasta raiz e selecione Adicionar Conta Compartilhada.

WebPhone

  • Escolha Opções - Telefone, após ter passado o mouse por cima da sua foto pessoal, na parte superior direita, para editar configurações do WebPhone.
  • Use a listagem de contatos do sistema de mensagens instantâneas, podendo clicar com o botão da direita em usuários para realizar chamadas.
  • Use o botão de telefone, ao topo, para realizar chamadas de áudio. O botão de conferência (monitor) é usado para realizar vídeochamadas.

Nota: O WebPhone requer um navegador que suporte WebRTC (atualmente Chrome e Firefox) ou plugin (para Internet Explorer e Safai). Você será redirecionado a uma página de ajuda, caso seu navegador não consiga gerenciar chamadas de áudio.

SmartAttach

Para economizar banda e ter controle sobre o conteúdo de emails enviados, você pode usar o SmartAttach, que substitui anexos por links para acesso aos arquivos no servidor.
  • Redija um novo e-mail normalmente, anexe os arquivos desejados e marque a opção "SmartAttach", no canto inferior direito de uma nova mensagem.
  • Navegue pela pasta Arquivos para visualizar e gerenciar anexos de e-mails armazenados na pasta.
  • Múltiplos anexos podem ser enviados a partir do seu Disco Local, Anexados a partir de Itens do Cliente Web (contatos, notas etc.) ou arrastando e soltando.

Mensagens de texto (SMS)

  • Selecione Novo(a) - SMS na barra de menu
  • Informe um número de telefone ou clique no ícone do livro de endereços para escolher destinatários que possuem um telefone celular.
  • Redija a mensagem incluindo o corpo e/ou assunto.
  • Verifique a mensagem no painel de visualização na parte inferior.

Itens recuperados

Esse recurso funciona da mesma maneira que a pasta Lixeira usada para mensagens de e-mail. Você pode ver os contatos, eventos, tarefas, etc. que foram excluídos. O administrador predefiniu por quantos dias os itens serão mantidos após a exclusão.
  • Navegue até a pasta Itens Recuperados, na árvore de pastas do lado esquerdo.
  • Clique com o botão direito no item e escolha Recuperar, para restaurar o item em seu local original.
  • Opcionalmente, você poderá excluir o item definitivamente dentro da pasta de Itens Recuperados.

Backup e restauração

Os dados de GroupWare são copiados para um backup no servidor automaticamente, mas você também pode criar seus próprios backups que também podem ser importados em outros aplicativos.
  • Opções - Importar/Exportar traz opções para fazer backup e restaurar todos os dados de GroupWare.
  • Alternativamente, você pode exportar toda a pasta de contatos em vários formatos.
  • Na pasta Contatos, selecione alguns itens, clique com botão direito e escolha Salvar como vCard.
  • No modo de exibição Lista do calendário, selecione alguns itens, clique com o botão direito e salve como vCalendar.
