IceWarp Веб-Клиент - это набор веб-приложений для повышения производительности, включая электронную почту, календари, документы, электронные таблицы, презентации, ГрупповойЧат, мгновенные сообщения, голосовые и видеозвонки и безопасное онлайн-хранилище.

Дизайн 21го века. Разработано с использованием новейших технологий:
  • Стандарт HTML5
  • WebRTC для аудио, видео звонков и совместного использования экрана
  • ONLYOFFICE проект, разработанный Ascensio System SIA
  • WeatherUnderground
  • Google Maps
  • WebAPI и TeamChatAPI для индивидуальной интеграции
