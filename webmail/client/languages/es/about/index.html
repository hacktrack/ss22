IceWarp WebClient es una suite completa de aplicaciones web de productividad, incluyendo correo electrónico, calendario, documentos, hojas de cálculo, presentaciones, TeamChat, chat privado, voz y video llamadas, y almacenamiento seguro en línea.

Diseñado para el siglo XXI. Desarrollado utilizando las últimas tecnologías:
  • Estándar HTML5 para una experiencia de clase de escritorio en la web
  • WebRTC para audio multipunto, transmisión de video y uso compartido de pantalla
  • ONLYOFFICEun proyecto desarrollado por Ascensio System SIA
  • WeatherUnderground
  • Google Maps
  • WebAPI and TeamChatAPI integraciones personalizadas
