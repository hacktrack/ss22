IceWarp WebClient esprime tutto il potenziale del Server di Messaggistica IceWarp:

  • progettato per offrire un'interfaccia tipo desktop (trascinamento, tasto destro...)
  • riunisce la tua casella di posta, i calendari, i contatti, le attività, le note e i file
  • consente la condivisione, la collaborazione, l'archiviazione dei file, la pianificazione delle riunioni e altro ancora
  • il WebIM integrato offre funzionalità di instant messaging e trasferimento file
  • WebPhone integrato per telefonia Internet e integrazione con PBX
  • funzionalità WebSMS (quando disponibili) per una comoda gestione dei messaggi di testo
